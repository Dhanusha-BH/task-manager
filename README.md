# 📝 Task Manager (React App)

A simple, clean, and interactive **Task Manager** built using React.
This app helps users manage their daily tasks efficiently with an easy-to-use interface.

---

## 🚀 Live Demo

👉 https://dhanusha-bh.github.io/task-manager/

---

## ✨ Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 🔄 Instant UI updates (React state)
* 📱 Responsive and clean design

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6)
* HTML5
* CSS3

## ⚙️ Installation & Setup

Clone the repository and run locally:

```bash
git clone https://github.com/dhanusha-bh/task-manager.git
cd task-manager
npm install
npm start
```

---

## 🌐 Deployment

This project is deployed using **GitHub Pages**.

To deploy:

```bash
npm run deploy
```
